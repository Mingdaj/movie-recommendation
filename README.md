项目环境：jdk8 + mysql5.7 + maven3.6.1 + node.js12

项目部署
前台：
1. 创建数据库，执行sql；
2. 将后端代码导入IDEA，下载依赖，修改数据库相关配置；
3. 配置tomcat，启动服务（保留项目名）。

后台：
1. 进入Movie-recommendation-system\src\main\webapp\admin目录下，cmd进入命令行；
2. 执行命令：npm install;
3. 执行命令：npm run serve。
